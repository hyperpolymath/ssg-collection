# About Orbital-SSG

Orbital-SSG is part of the poly-ssg ecosystem, demonstrating static site generation in Grain compiled to WebAssembly.

## Why Grain/WebAssembly?

Grain is a modern functional language that compiles to WebAssembly, offering type safety with ML-style syntax. Its WASM output enables portable, sandboxed execution across platforms. Orbital-SSG demonstrates how functional programming and WebAssembly can deliver fast, secure static site generation.
