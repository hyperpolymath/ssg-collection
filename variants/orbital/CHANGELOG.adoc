= Changelog

All notable changes to this project will be documented in this file.

== [Unreleased]

=== Added

- Justfile with comprehensive task automation
- Mustfile with mandatory quality checks
- cookbook.adoc with CLI recipes and combinatorics
- PLAYBOOK.scm with development workflows
- AGENTIC.scm with agent capabilities
- NEUROSYM.scm with symbolic structure

- Container support with Podman/Docker
- .editorconfig for consistent formatting
- .tool-versions for asdf version management

== [0.2.0] - 2025-12-22

=== Added
- Input sanitization for Julia string interpolation
- Input sanitization for Lisp string interpolation
- Identifier validation for Julia module names
- Comprehensive roadmap in STATE.scm

=== Security
- Fixed code injection vulnerability in staticwebpages.js
- Fixed code injection vulnerability in documenter.js
- Fixed code injection vulnerability in franklin.js
- Fixed code injection vulnerability in coleslaw.js

=== Changed
- Updated META.scm with new ADRs
- Updated ECOSYSTEM.scm with expanded relationships
- Updated STATE.scm with 44/44 component tracking
- Replaced template placeholders in SECURITY.md

== [0.1.0] - 2025-12-17

=== Added
- Initial release with 28 SSG adapters
- Support for 17 programming languages
- MCP-compatible tool exports
- RSR compliance configuration
- CodeQL security scanning
- Dependabot dependency updates

=== Adapters
- Rust: Cobalt, mdBook, Zola
- Haskell: Ema, Hakyll
- Julia: Documenter.jl, Franklin.jl, StaticWebPages.jl
- Clojure: Babashka, Cryogen, Perun
- Elixir: NimblePublisher, Serum, Tableau
- Scala: Laika, ScalaTex
- Racket: Frog, Pollen
- Common Lisp: Coleslaw
- F#: Fornax
- Crystal: Marmot
- Nim: Nimrod
- Kotlin: Orchid
- D: Reggae
- Swift: Publish
- Tcl: Wub
- OCaml: YOCaml
- Erlang: Zotonic
