# About my-ssg

my-ssg is part of the poly-ssg ecosystem, demonstrating static site generation in Janet.

## Why Janet?

Janet is a lightweight, embeddable Lisp with a focus on simplicity and C interop. It compiles to a small binary with no dependencies, making it perfect for portable tooling. my-ssg leverages Janet's powerful macro system and PEG pattern matching for elegant markdown parsing.
