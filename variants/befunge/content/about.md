# About befunge-ssg

befunge-ssg is part of the poly-ssg ecosystem, demonstrating static site generation in Befunge-93.

## Why Befunge-93?

Befunge is a two-dimensional esoteric programming language where the program counter can move in any direction on a grid. This unique paradigm challenges conventional thinking about program flow and demonstrates that even non-linear code can produce practical results.
