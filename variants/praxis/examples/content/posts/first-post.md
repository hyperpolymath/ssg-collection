# My First Post

This is my first blog post using doit-ssg.

## Why Static Sites?

Static sites are:

1. **Fast** - No server-side rendering
2. **Secure** - No database to hack
3. **Cheap** - Can be hosted anywhere
4. **Reliable** - No moving parts

## Conclusion

Static site generators are great for blogs, documentation, and marketing sites.
