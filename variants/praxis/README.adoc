A satellite SSG (Static Site Generator) implementation in the hyperpolymath ecosystem, integrating with the poly-ssg-mcp hub to provide unified access to 28 static site generators across 14+ programming languages.

image:https://img.shields.io/badge/RSR-Gold-gold[RSR Gold]
image:https://img.shields.io/badge/adapters-28-blue[28 Adapters]
image:https://img.shields.io/badge/License-PMPL--1.0-blue.svg[License: PMPL-1.0,link="https://github.com/hyperpolymath/palimpsest-license"]

== Overview

praxis-ssg is a satellite project that consumes SSG adapters from the https://github.com/hyperpolymath/poly-ssg-mcp[poly-ssg-mcp] hub, providing a unified interface to build static sites using any of 28 supported generators.

=== Supported SSG Languages

[cols="1,3"]
|===
|Tier |Languages & SSGs

|**Tier 1**
|Rust (Zola, Cobalt, mdBook), Haskell (Hakyll, Ema), Elixir (Serum, NimblePublisher, Tableau)

|**Tier 2**
|Julia (Franklin, Documenter, StaticWebPages), Clojure (Cryogen, Perun, Babashka), Racket (Frog, Pollen), Scala (Laika, ScalaTex)

|**Tier 3**
|Kotlin (Orchid), F# (Fornax), OCaml (YOCaml), Nim (Nimrod), Crystal (Marmot), Swift (Publish), D (Reggae), Tcl (Wub), Erlang (Zotonic), Common Lisp (Coleslaw)
|===

== Quick Start

=== Prerequisites

* https://deno.land/[Deno] runtime (v1.x+)
* https://github.com/casey/just[just] task runner
* Git

=== Installation

[source,bash]
----
# Clone the repository
git clone https://github.com/hyperpolymath/praxis-ssg.git
cd praxis-ssg

# Verify setup
just status

# List available adapters
just adapters
----

=== Basic Usage

[source,bash]
----
# Run all tests
just test

# Check adapter syntax
just check-syntax

# Verify adapter interface
just adapter-verify

# Full CI pipeline
just ci
----

== Adapters

[source,javascript]
----
// Adapter interface
export const name = "SSGName";
export const language = "Language";
export const description = "Brief description";

export async function connect() { /* ... */ }
export async function disconnect() { /* ... */ }
export function isConnected() { /* ... */ }

export const tools = [
  {
    name: "tool_name",
    description: "What the tool does",
    inputSchema: { /* JSON Schema */ },
    execute: async (params) => { /* ... */ }
  }
];
----

=== Adapter Commands

[source,bash]
----
# List all adapters
just adapters

# Count adapters
just adapter-count

# Verify interface compliance
just adapter-verify

# Sync from hub
just adapter-sync
----
