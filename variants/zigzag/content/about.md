# About zigzag-ssg

zigzag-ssg is part of the poly-ssg ecosystem, demonstrating static site generation in Zig.

## Why Zig?

Zig offers manual memory management without garbage collection, making it ideal for high-performance tooling. Its comptime features enable zero-cost abstractions, and the language's simplicity makes the codebase easy to understand and maintain. Zig's excellent C interoperability also opens doors for leveraging existing markdown parsing libraries.
