== Language Tier Classification

=== Tier 1 Languages (Preferred)
* Rust
* Elixir
* Zig
* Ada
* Haskell
* ReScript

=== Tier 2 Languages (Acceptable)
* Nickel (configuration)
* Racket (scripting)
* Guile Scheme (state management)
* Nix (derivations)

=== Special Cases
* Conway's Game of Life - Primary language for hackenbush-ssg SSG logic
* TypeScript/Deno - Allowed ONLY in runtime/ for host simulator (no SSG logic)

=== Restricted Languages
* Python - Only allowed in salt/ directories for SaltStack
* JavaScript - Not allowed for SSG logic
* CUE - Not permitted, use Nickel or Guile
