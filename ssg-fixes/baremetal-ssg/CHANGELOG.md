# Changelog

All notable changes to baremetal-ssg will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-12-22

### Added
- 28 SSG adapters across 18 programming languages
- Complete SCM metadata files (STATE, META, ECOSYSTEM, ROADMAP, PLAYBOOK, AGENTIC, NEUROSYM, AI)
- Justfile with 20+ task runner recipes
- Mustfile with mandatory quality checks
- Comprehensive cookbook.adoc with CLI, Nickel, and Just recipes
- CI/CD pipeline with GitHub Actions
- Containerfiles for standard and Alpine variants
- Git hooks for pre-commit and pre-push validation
- Unit and integration test framework
- RSR Gold compliance

### Adapters by Language
- **Rust**: Zola, Cobalt, mdBook
- **Haskell**: Hakyll, Ema
- **Elixir**: Serum, Tableau
- **Clojure**: Cryogen, Perun
- **Julia**: Franklin, Documenter
- **Racket**: Frog, Pollen
- **OCaml**: YOCaml
- **F#**: Fornax
- **Scala**: Laika, ScalaTex
- **Common Lisp**: Coleslaw
- **Nim**: Nimrod, Nimble-Publisher
- **D**: Reggae
- **Crystal**: Marmot
- **Swift**: Publish
- **Erlang**: Zotonic, Wub
- **Tcl**: StaticWebPages
- **Babashka**: Babashka
- **Kotlin**: Orchid

### Security
- Resolved all SECURITY.md template placeholders
- Added CodeQL analysis workflow
- Added Dependabot configuration
- Implemented security checks in CI pipeline
