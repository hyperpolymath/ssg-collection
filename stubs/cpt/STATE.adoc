== Description

A pure QPL (Quantum Programming Language) static site generator, demonstrating quantum-classical hybrid computation for document transformation. This project is a satellite of the poly-ssg-mcp ecosystem.

== Dependencies

* qpl-toolchain (QPL compiler and QRAM simulator)
* Guix (package management)
