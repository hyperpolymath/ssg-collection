# About Rescribe-SSG

Rescribe-SSG is part of the poly-ssg ecosystem, demonstrating static site generation in ReScript.

## Why ReScript?

ReScript offers sound typing, fast compilation, and readable JavaScript output. Its OCaml heritage provides pattern matching and algebraic data types while compiling to efficient, human-readable JS. Rescribe-SSG leverages ReScript's type safety to catch content and template errors at compile time.
