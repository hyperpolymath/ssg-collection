# About Parallax-SSG

Parallax-SSG is part of the poly-ssg ecosystem, demonstrating static site generation in Chapel.

## Why Chapel?

Chapel is a parallel programming language designed for productive supercomputing. Its elegant syntax and built-in parallelism make it uniquely suited for processing large content collections simultaneously. Parallax-SSG leverages Chapel's data parallelism for blazing-fast multi-file builds.
