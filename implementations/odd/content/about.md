# About Odd-SSG

Odd-SSG is part of the poly-ssg ecosystem, demonstrating static site generation in Forth.

## Why Forth?

Forth's stack-based, concatenative nature provides an unconventional yet powerful approach to text processing. Its minimal runtime and direct hardware access make it incredibly efficient. Odd-SSG explores how Forth's unique paradigm can be applied to content transformation pipelines.
