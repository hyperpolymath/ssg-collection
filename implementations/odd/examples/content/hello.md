# Hello World

Welcome to **odd-ssg**, the satellite SSG adapter provider.

## Features

- 30 SSG adapters
- MCP protocol support
- Mill-based synthesis engine
- Accessibility support (BSL, ASL, GSL, Makaton)

## Getting Started

```bash
# Initialize with your favorite SSG
just mcp

# Build your site
just build
```
